# INSA's Grade Rain Of Death
Projet universitaire, réalisé à l'INSA Toulouse, de développement d'un jeu vidéo Web en JavaScript.

Le joueur incarne un étudiant qui se déplace horizontalement afin de collecter des bonnes notes tombant du haut de l’écran, tout en évitant les mauvaises notes qui font perdre de la vie.

Le jeu propose :

- un mode un joueur ou deux joueurs,

- un support clavier et tactile,

- un système de score et de vie,

- une difficulté progressive,

- des animations visuelles et sonores,

- un classement des meilleurs scores stocké sur la session.
